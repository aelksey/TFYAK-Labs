# Ответы на контрольные вопросы

### 1. Что такое терминальные и нетерминальные символы формальных грамматик?

**Терминальные символы (Aₜ)** — это элементарные символы языка, из которых строятся предложения. Они не могут быть заменены другими символами по правилам грамматики. Примеры: идентификаторы, константы, знаки операций `+`, `-`, `*`, `/`, точка с запятой.

**Нетерминальные символы (Aₙ)** — это вспомогательные символы, используемые только в метаязыке для описания синтаксических конструкций. Они могут быть заменены на другие символы (как терминальные, так и нетерминальные) по правилам грамматики. Примеры: `<выражение>`, `<оператор>`, `<условие>`.

---

### 2. Что такое рекурсия? Перечислите виды рекурсии.

**Рекурсия** — это свойство нетерминала, когда из него может быть выведена цепочка, содержащая этот же нетерминал.

**Виды рекурсии:**
- **Прямая рекурсия** — нетерминал в правой части правила сразу ссылается на себя: `S : S + T`
- **Косвенная рекурсия** — нетерминал ссылается на себя через цепочку правил: `S : T`, `T : S + U`
- **Левая рекурсия** — нетерминал находится в начале правой части: `S : S α`
- **Правая рекурсия** — нетерминал находится в конце правой части: `S : α S`
- **Общая рекурсия** — нетерминал находится внутри правой части: `S : α S β`

---

### 3. Что такое контекстно-свободные грамматики?

**Контекстно-свободные грамматики (КС-грамматики)** — это грамматики типа 2 по классификации Хомского, у которых левая часть каждого правила состоит ровно из одного нетерминала.

**Особенности:**
- Замена нетерминала не зависит от окружающего контекста (символов слева и справа)
- Позволяют описывать большинство конструкций языков программирования
- Для них существуют эффективные алгоритмы синтаксического анализа
- Форма записи: `N : α`, где N — один нетерминал, α — цепочка из терминалов и нетерминалов

---

### 4. Вычислите множество предшественников цепочки символов W R для грамматики Gₐ₂.

*Для ответа на этот вопрос необходима грамматика Gₐ₂ из документа (табл. 6).*

**Общая формула для вычисления множества предшественников цепочки:**

Если цепочка β = X₁X₂...Xₙ, то:
- Mпр(β) = Mпр(X₁), если X₁ — неаннулируемый символ
- Mпр(β) = Mпр(X₁) ∪ Mпр(X₂...Xₙ), если X₁ — аннулируемый нетерминал

### 5. Является ли однозначной грамматика:

```
S : S | S
S : S S
S : ( S )
S : ident
```

**Ответ:** Нет, эта грамматика **неоднозначна**.

**Причина:**
- Первое правило `S : S | S` эквивалентно `S : S` (так как выбор между одинаковыми вариантами бессмыслен), что создает левую рекурсию
- Второе правило `S : S S` допускает разные способы группировки для одного и того же предложения
- Для предложения `ident ident ident` можно построить разные деревья разбора (левоассоциативное и правоассоциативное)

---

### 6. Что такое аннулируемый нетерминал?

**Аннулируемый нетерминал** — это нетерминальный символ, из которого может быть выведена пустая цепочка ε (цепочка нулевой длины).

**Примеры:**
- Если есть правило `A : ε`, то A аннулируем
- Если `A : B C`, а B и C аннулируемы, то A аннулируем
- Если `A : B`, а B аннулируем, то A аннулируем

**Важно:** Аннулируемые нетерминалы влияют на вычисление множеств предшественников и последователей.

---

### 7. Что такое стековая память? Какие операции обычно определяются над стековой памятью?

**Стековая память** — это структура данных, работающая по принципу LIFO (Last In — First Out), то есть последний занесённый элемент извлекается первым.

**Основные операции:**
- **Push (занесение)** — поместить элемент на вершину стека
- **Pop (снятие)** — удалить элемент с вершины стека
- **Top (чтение)** — прочитать элемент на вершине без удаления
- **IsEmpty (проверка пустоты)** — определить, пуст ли стек

В синтаксических анализаторах стек используется для хранения состояний автомата или символов грамматики при разборе.

---

### 8. Что такое множество последователей символа грамматики?

**Множество последователей** символа X (обозначается Mпосл(X)) — это множество терминальных символов, которые могут появляться непосредственно справа от X в какой-либо цепочке, выводимой из начального нетерминала грамматики.

**Правила вычисления:**
1. Если есть правило `N : α X β`, то все символы из Mпр(β) входят в Mпосл(X)
2. Если есть правило `N : α X β`, и β аннулируема, то Mпосл(N) ⊆ Mпосл(X)
3. Для начального нетерминала S: псевдотерминал ► ∈ Mпосл(S)

---

### 9. Чем понятие вывода отличается от понятия непосредственного вывода?

| Характеристика | Непосредственный вывод | Вывод |
|----------------|------------------------|-------|
| **Обозначение** | σ → ψ | σ ⇒ ψ |
| **Количество шагов** | Ровно один шаг | Один или более шагов |
| **Применение правил** | Одно правило, одна замена | Последовательность правил |
| **Длина цепочки** | Может увеличиваться или уменьшаться | Может изменяться многократно |

**Пример:**
- Непосредственный вывод: `S + T → S + const` (применили правило `T : const`)
- Вывод: `S ⇒ S + T ⇒ S + const` (два шага)

---

### 10. Что такое недетерминированность конечного автомата без памяти? Какие бывают виды недетерминированности?

**Недетерминированность КА** — ситуация, когда для одного и того же состояния и входного символа существует несколько возможных переходов, либо есть переход по пустой цепочке ε.

**Виды недетерминированности:**

| Вид | Описание |
|-----|----------|
| **Первого рода** | Наличие перехода по пустой цепочке ε из рабочего состояния в другое рабочее состояние |
| **Второго рода** | Наличие нескольких переходов из одного состояния по одному и тому же входному символу в разные состояния |

---

### 11. Устраните левую рекурсию в такой грамматике:

```
S : ( L )
S : ident
L : L , S
L : S
```

**Исходная грамматика:**
- S : ( L )
- S : ident
- L : L , S
- L : S

**Проблема:** L : L , S — прямая левая рекурсия.

**Устранение левой рекурсии** (общая формула: A → Aα | β заменяем на A → βA', A' → αA' | ε):

Для L: L → S | L , S

Заменяем:
- L → S L'
- L' → , S L' | ε

**Результат (грамматика без левой рекурсии):**
```
S : ( L )
S : ident
L : S L'
L' : , S L'
L' : ε
```

### 12. Что такое дерево грамматического разбора?

**Дерево грамматического разбора** — это графическое представление процесса вывода предложения в формальной грамматике.

**Свойства:**
- **Корень** — начальный нетерминал S
- **Внутренние узлы** — нетерминалы, к которым применяются правила
- **Листья** — терминальные символы (в порядке, соответствующем предложению)
- **Дуги** — показывают применение правил подстановки

**Пример для выражения `a + b` в грамматике `S → S + T | T, T → id`:**
```
      S
     /|\
    S + T
    |   |
    T   id(b)
    |
    id(a)
```

---

### 13. Какие нетерминальные символы называются бесплодными?

**Бесплодный нетерминал** — это нетерминальный символ, из которого нельзя вывести цепочку, состоящую только из терминальных символов.

**Признаки:**
- Нетерминал не имеет правил, правая часть которых состоит только из терминалов
- Все правила для этого нетерминала ведут к другим бесплодным нетерминалам
- Из него нельзя породить ни одного слова языка

**Пример:**
```
A : B C
B : B
C : A
```
Здесь A, B, C — бесплодны, так как нет ни одного правила, порождающего терминалы.

---

### 14. Что такое рекурсивный цикл в грамматике?

**Рекурсивный цикл** — это ситуация, когда существует последовательность правил, позволяющая вывести из нетерминала цепочку, содержащую этот же нетерминал, без продвижения к терминальным символам.

**Пример:**
```
A : B
B : C
C : A
```
Это косвенная рекурсия, образующая цикл.

**Проблема:** Рекурсивные циклы могут привести к бесконечному выводу при отсутствии терминальных правил.

---

### 15. Какие грамматики называются эквивалентными?

Две грамматики называются **эквивалентными**, если они порождают один и тот же язык, то есть множество правильных предложений у них совпадает.

**Пример:** В документе приведены грамматики Gₐ₁ и Gₐ₂, которые порождают один и тот же язык арифметических выражений, но имеют разную структуру правил.

---

### 16. Могут ли быть эквивалентными два конечных автомата без памяти, имеющие разное количество финальных состояний?

**Да, могут.**

**Обоснование:** Эквивалентность автоматов определяется не количеством финальных состояний, а множеством распознаваемых цепочек (языком). Два автомата могут распознавать один и тот же язык, имея разное количество финальных состояний.

**Пример:**
- Автомат А: одно финальное состояние, распознающее цепочки, заканчивающиеся на `0`
- Автомат В: два финальных состояния, но распознающие те же цепочки

### 17. Вычислите множества последователей для грамматики из вопроса 6.5.

*Для вычисления необходима полная грамматика из вопроса 6.5. Формулы для вычисления:*

**Правила вычисления Mпосл(X):**
1. Если есть правило `N : α X β`, то Mпр(β) ⊆ Mпосл(X)
2. Если есть правило `N : α X β` и β аннулируема, то Mпосл(N) ⊆ Mпосл(X)
3. Для начального нетерминала S: ► ∈ Mпосл(S)

### 18. Что такое контекстно-зависимая грамматика?

**Контекстно-зависимая грамматика** — грамматика типа 1 по классификации Хомского, у которой левая часть правила может содержать несколько символов, но длина правой части не меньше длины левой.

**Особенности:**
- Применение правила зависит от контекста (символов слева и справа)
- Правила имеют вид αAβ → αγβ, где A — нетерминал, γ — непустая цепочка
- Не может уменьшать длину цепочки (кроме правила S → ε)

**Пример:** `x A y → x B C y` — замена A на BC разрешена только в окружении x и y.

---

### 19. Что такое отношение предшествования символов?

**Отношение предшествования** — это отношение между символами грамматики, определяющее, какие символы могут находиться в начале цепочки, выводимой из данного символа.

**Множество предшественников** Mпр(X) для символа X — это множество терминалов, с которых могут начинаться цепочки, выводимые из X.

**Свойства:**
- Любой символ является предшественником самого себя (для вывода длины 0)
- Если X — терминал, то Mпр(X) = {X}
- Если X — нетерминал, то Mпр(X) = объединение Mпр(α) для всех правил X → α

---

### 20. Можно ли удалить из грамматики пряморекурсивный нетерминал?

**Можно, но не удалением, а преобразованием.**

Прямая левая рекурсия устраняется заменой правил:
- Исходные: `A → Aα | β`
- Результат: `A → βA'`, `A' → αA' | ε`

Прямая правая рекурсия:
- Исходные: `A → αA | β`
- Результат: `A → βA'`, `A' → αA' | ε` (слева направо)

**Важно:** Удаление рекурсии — это эквивалентное преобразование, сохраняющее порождаемый язык.

### 22. Что такое факторизация?

**Факторизация** — это преобразование грамматики, устраняющее неоднозначность, вызванную наличием у нетерминала нескольких правил с общим префиксом в правой части.

**Пример:**
- Исходные правила: `A → αβ | αγ`
- После факторизации: `A → αA'`, `A' → β | γ`

**Цель:** Получение LL(1)-грамматики, пригодной для нисходящего синтаксического анализа.

### 23. Сформулируйте физический смысл перехода по пустой цепочке в финальное состояние конечного автомата без памяти.

**Физический смысл:** Переход по пустой цепочке ε в финальное состояние эквивалентен следующей последовательности операций:
1. Чтение одного символа с входа
2. Обнаружение того, что символ не принадлежит текущему слову
3. Возврат символа на вход автомата

Это позволяет автомату «заглянуть» на один символ вперед, чтобы определить, закончилось ли текущее слово и начинается ли следующее.

### 24. Можно ли и, если можно, то как преобразовать косвенную рекурсию в прямую?

**Можно.** Косвенная рекурсия преобразуется в прямую путем подстановки правил.

**Пример:**
```
A → B α
B → A β | γ
```

Подставляем B в A:
```
A → (A β | γ) α
A → A β α | γ α
```

Получили прямую левую рекурсию для A:
```
A → A β α | γ α
```

### 25. Определите алфавиты терминальных и нетерминальных символов для грамматики из вопроса 6.11.

*Для точного ответа необходима грамматика из вопроса 6.11. Общий подход:*

**Терминалы (Aₜ)** — все символы, которые могут появляться в конечных предложениях:
- Идентификаторы (ident)
- Константы (const)
- Знаки операций (+, -, *, /, =, <, >, и т.д.)
- Разделители (;, ,, ., (, ), {, })
- Ключевые слова (if, else, while, и т.д.)

**Нетерминалы (Aₙ)** — все символы, используемые в левых частях правил:
- S (начальный нетерминал)
- Выражение, оператор, условие, блок и т.д.

### 26. Какие конечные автоматы без памяти называются эквивалентными?

Два конечных автомата без памяти называются **эквивалентными**, если они распознают один и тот же язык, то есть множество входных цепочек, приводящих к останову в финальном состоянии, у них совпадает.

**Критерии:**
- Для любой входной цепочки оба автомата либо останавливаются в финальном состоянии, либо нет
- Количество состояний и финальных состояний может различаться

### 27. Перечислите условия, которые определяют отношение последования символов грамматики.

Отношение последования определяется четырьмя условиями:

**Условие 1 (непосредственное следование):** В грамматике есть правило `N : ϕ X Y ψ`, где ϕ и ψ — произвольные цепочки.

**Условие 2 (непосредственное следование через аннулируемые):** В грамматике есть правило `N : ϕ X σ Y ψ`, где σ — цепочка из аннулируемых нетерминалов.

**Условие 3 (косвенное следование):** Y является последователем некоторого символа Z, и есть правило `Z : χ X σ`, где σ — из аннулируемых или пустая.

**Условие 4 (косвенное следование через цепочку):** Существует цепочка правил, связывающая Z с X через аннулируемые нетерминалы.

### 28. Вычислите вручную множества предшественников и последователей и все свойства символов для фрагмента грамматики, определяющего синтаксис условного оператора языка С:

```
Operator : ...
Operator : "if" "(" LogicalExpression ")" Operator Rest
Rest : "else" Operator
Rest : ε
```

**Свойства символов:**
- **Аннулируемость:** Rest — аннулируем (есть правило Rest : ε)
- **Недостижимость:** все символы достижимы из Operator
- **Бесплодность:** нет бесплодных символов

**Множества предшественников:**
- Mпр(Operator) = {"if", ...} (из первого правила)
- Mпр(Rest) = {"else", ...} из первого правила Rest, и { ... } из ε

**Множества последователей:**
- Mпосл(Rest) = {►} (после Rest может быть конец файла)

### 29. Является ли грамматика вопроса 6.29 однозначной?

Грамматика условного оператора языка С **неоднозначна**, так как существует проблема «висячего else» (dangling else).

**Пример неоднозначности:**
```
if (x) if (y) a = 1; else b = 2;
```
Это предложение может быть разобрано двумя способами:
1. `else` относится к первому `if`
2. `else` относится ко второму `if`

### 30. Как соотносятся между собой языки и грамматики?

| Отношение | Описание |
|-----------|----------|
| Грамматика определяет язык | Каждая грамматика порождает единственный язык |
| Язык может иметь несколько грамматик | Для одного языка могут существовать разные грамматики (эквивалентные) |
| Не всякая грамматика подходит для анализа | Некоторые грамматики не позволяют построить эффективный синтаксический анализатор |
| Язык шире грамматики | Грамматика — это формальное описание языка, а язык — множество всех правильных предложений |

### 31. Могут ли быть эквивалентными два конечных автомата без памяти, имеющие различное количество рабочих состояний?

**Да, могут.**

**Обоснование:** Эквивалентность определяется распознаваемым языком, а не количеством состояний. Один автомат может быть оптимизирован (иметь меньше состояний), а другой — нет, но распознавать те же цепочки.

**Пример:** Оптимальный и неоптимальный автоматы из документа (раздел об оптимальности КА) распознают один и тот же язык, но имеют разное количество состояний.

### 32. Как вычисляется множество предшественников цепочки символов?

**Алгоритм вычисления Mпр(β) для цепочки β = X₁X₂...Xₙ:**

1. Если n = 0 (пустая цепочка), то Mпр(ε) = ∅
2. Mпр(β) = Mпр(X₁)
3. Если X₁ — аннулируемый нетерминал, то Mпр(β) = Mпр(β) ∪ Mпр(X₂...Xₙ)
4. Повторять шаг 3 для всех аннулируемых символов в начале цепочки

**Формула из документа:**
Mпр(β) = Mпр(X₁) ∪ Mпр(X₂...Xₙ), если X₁ — аннулируемый нетерминал, иначе Mпр(β) = Mпр(X₁)

### 33. Почему из системы порождающих правил грамматики не может быть удален ее начальный нетерминал?

**Причины:**
1. Начальный нетерминал определяет язык грамматики — удаление правил для него изменит порождаемый язык
2. Синтаксический анализатор начинает разбор с начального нетерминала
3. Без начального нетерминала грамматика не может порождать правильные предложения

**Исключение:** Если начальный нетерминал бесплоден и недостижим, он может быть удален, но тогда грамматика становится пустой.

### 34. Как связаны между собой понятия вывода и дерева грамматического разбора?

| Вывод | Дерево разбора |
|-------|----------------|
| Линейная последовательность шагов | Графическое представление |
| Показывает порядок применения правил | Показывает иерархическую структуру |
| Может быть левым или правым | Единственно для однозначной грамматики |
| Каждому выводу соответствует дерево | Каждому дереву соответствует множество выводов |

**Связь:** Дерево разбора — это компактное представление всех возможных выводов (левого, правого и смешанного) для данного предложения.
