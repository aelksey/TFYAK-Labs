# Краткая теория по лабораторной работе № 3

## Формальные грамматики

**Формальная грамматика** — это математическая модель, описывающая синтаксис языка. Она определяется как совокупность:

**G = {Aₜ, Aₙ, S, P}**

где:
- **Aₜ** — алфавит терминальных символов (слова языка: идентификаторы, константы, операторы)
- **Aₙ** — алфавит нетерминальных символов (синтаксические конструкции: `<выражение>`, `<оператор>`)
- **S** — начальный нетерминал (главная синтаксическая конструкция)
- **P** — система правил подстановки (продукций) вида α : β

## Основные понятия

**Вывод** — последовательность замен нетерминалов по правилам грамматики.  
**Непосредственный вывод** (σ → ψ) — замена левой части правила на правую.  
**Дерево грамматического разбора** — графическое представление вывода.

## Классификация грамматик по Хомскому

1. **Регулярные** (тип 3) — правые части содержат не более одного нетерминала
2. **Контекстно-свободные** (тип 2) — левая часть состоит из одного нетерминала
3. **Контекстно-зависимые** (тип 1) — контекст влияет на применение правила
4. **Общего вида** (тип 0) — без ограничений

Для трансляторов используются **контекстно-свободные грамматики (КС-грамматики)**.

## Свойства символов грамматики

| Свойство | Определение |
|----------|-------------|
| **Аннулируемость** | Из нетерминала можно вывести пустую цепочку ε |
| **Недостижимость** | Символ не появляется в цепочках, выводимых из S |
| **Бесплодность** | Из нетерминала нельзя вывести цепочку только из терминалов |

## Отношения предшествования и следования

**Множество предшественников** символа X — символы, с которых начинаются цепочки, выводимые из X.

**Множество последователей** символа X — символы, которые могут следовать за X в выводимых цепочках.

## Свойства грамматик

**Рекурсивность** — нетерминал называется рекурсивным, если из него может быть выведена цепочка, содержащая этот же символ. Грамматика называется рекурсивной, если рекурсивен хотя бы один нетерминал.

**Однозначность** — грамматика однозначна, если каждое правильное предложение имеет единственное дерево разбора.
